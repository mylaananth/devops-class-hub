

# KEC Collab GUVI — DevOps Class Website

## Overview
A promotional and informational website for a DevOps class organized by KEC in collaboration with GUVI. The site will have a modern, dark techy theme fitting the DevOps domain, with GUVI-inspired accent colors (blue/green).

---

## Page 1: Home / Landing Page
- **Hero Section** — Bold headline like "Master DevOps for the Modern Industry", subtitle about KEC × GUVI collaboration, and a CTA button ("Register Now" / "Explore Curriculum")
- **Key Highlights** — Cards showcasing: CI/CD, Docker, Kubernetes, Cloud Deployment, Monitoring & more
- **Why DevOps?** — Brief section on industry demand and career opportunities
- **About the Collaboration** — Logos and short description of KEC and GUVI partnership
- **Testimonials / Stats** — Placement stats or student feedback (placeholder data)
- **Footer** — Links, social media, contact info

## Page 2: Curriculum & Registration
- **Curriculum / Syllabus Section** — Timeline or accordion listing DevOps topics covered (Git, Linux, Docker, Kubernetes, Jenkins, AWS basics, etc.)
- **Class Details** — Duration, schedule, mode (online/offline), prerequisites
- **Registration Form** — Name, email, phone, college year, and submit button (frontend only, no backend)
- **FAQ Section** — Common questions about the class

---

## Design & Style
- Dark background with green/blue accents inspired by GUVI branding
- Terminal/code-style design touches (monospace fonts for headings, subtle grid patterns)
- Fully responsive for mobile and desktop
- Smooth scroll navigation between sections
- Navigation bar with links to both pages
